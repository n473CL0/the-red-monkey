# The Red Monkey Website Guide

Think of this place as a shared online folder where your website lives. You can change text and links here, and the system will update the live website for you.

## How to Update the About Page

1. Open the file named `about.md`.
2. Click the pencil icon in the top right corner to edit the document.
3. Ignore the code between the dashes at the top. Scroll down to change the regular text and the opening hours.
4. Click the green "Commit changes" button to save your work. Think of "commit" as your final "save."

## How to Update Contact Details and Links

1. Open the file named `_config.yml`.
2. Click the pencil icon to edit.
3. Find the `contact:` section to change your phone number, address, or map link.
4. Find the `links:` section to change your Facebook, Instagram, or beer board links.
5. Type your new details inside the quote marks. **Do not delete the quote marks.**
6. Click the green "Commit changes" button to save your work.

## The Wait

When you save a file, the changes do not happen instantly. The system needs a few minutes to build the new pages. Wait three to five minutes, then refresh your live website to see the updates.
